ce laboratoire est fait pour python3 x64. pour le lancer, utiliser la commande python Lab3.py .

lab3.py utilise le module spam.pyd qui provient de la dernière version du code du Lab2.



Fonctionnement du programme:

après avoirt été lancé, une fenêtre apparait avec les six boutons suivant:

1- run: permet de choisir une vidéo à jouer
2- play: joue la vidéo
3- pause: pause la vidéo
4- replay: relance la vidéo en train de jouer
5- 1x: met la vitesse de la vidéo à 1x
6- 2x: met la vitesse de la vidéo à 2x
	
le X dans le coin haut droit de l'écran permet également de fermer la fenêtre et d'arrêter la vidéo.
